## Công nghệ & Thuật toán

Dự án tập trung vào việc cài đặt và tối ưu các thuật toán cốt lõi của AI:

* **Ngôn ngữ:** Python
* **Thư viện:** NumPy, Matplotlib (để trực quan hóa đường đi/đồ thị).
* **Thuật toán nổi bật:**
    * Breadth-First Search (BFS), Depth-First Search (DFS).
    * A* Search.
    * Hill Climbing, Simulated Annealing (cho bài toán TSP).
    * Genetic Algorithms (Giải thuật di truyền).
